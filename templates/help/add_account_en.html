📘 Account Creation Guide (KR Accounting Standards)

1️⃣ Step One: Select the Account Type

According to the accounting standards of the Kyrgyz Republic, all accounts are classified into five main categories. Selecting the correct account type ensures accurate financial reports (Balance Sheet, Profit & Loss).

  • Asset: Company resources (cash, bank accounts, inventory, buildings).
  • Liability: Company obligations (payables, loans, taxes payable).
  • Equity: Owners’ capital and retained earnings.
  • Income: Revenue from sales and other business activities.
  • Expense: Costs incurred to operate the business.
2️⃣ Step Two: Select the Account Subtype

⚠️ This field is critical for the system.
For AI analysis and automated transactions to work correctly, the account subtype must be selected.

  • Cash: For accounts holding physical cash.
  • Bank: For bank accounts (balances are tracked separately).
  • Tax: For all tax-related accounts (VAT, income tax, etc.).
  • Other: For all other accounts (inventory, rent, etc.).
3️⃣ Step Three: Use the KR Standard Account Code

In KR accounting, each account has a unique numeric code. It is recommended to use these standard codes when creating accounts.

Account Name (KR Standard) Code Type Subtype
Cash (KGS) 1110 Asset cash
Bank Account (KGS) 1210 Asset bank
Inventory for Sale 1610 Asset other
Accounts Payable (Suppliers) 3110 Liability other
VAT Payable 3430 Liability tax
Share Capital 5110 Equity other
Revenue from Product Sales 6110 Income other
General Administrative Expenses 8000 Expense other
4️⃣ Step Four: Currency Separation

According to NBKR regulations, local currency (KGS) and foreign currencies must not be mixed within the same account.

  • For foreign currency cash, create a separate account (e.g. 1120 — Foreign Currency Cash).
  • During transactions, the system automatically converts amounts using the NBKR exchange rate and reflects them in reports.
5️⃣ Step Five: Add the Account via the Interface (UI Guide)
  1. Account Name: Enter the account name (e.g. “Main Cash”).
  2. Account Type: Select the account type.
  3. Account Subtype: Select the required subtype.
  4. Currency: Select the account currency.
  5. Code: Enter the KR standard account code (e.g. 1110).
  6. Save: Click the Save button.
Important Notes:
  • When a company is created, core accounts (Cash, Bank, Revenue, Expenses) are generated automatically.
  • Accounts are isolated per company (multi-tenancy) and are not visible to other companies.
  • If an account is temporarily not used, it is recommended to set is_active = False instead of deleting it.
